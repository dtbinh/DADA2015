\subsubsection{Milestones}
 
The project will be organized in four main phases, separated by three milestones at $T_0+12$, $T_0+24$ and $T_0+36$:

\begin{description}
\item Phase 1 ($T_0 \rightarrow T_0+12$) : Specifications and scenarios.
\item Milestone 1 ($T_0+12$): Deliver specifications and scenarios.
\item Phase 2 ($T_0+12 \rightarrow T_0+24$) : First prototype.
\item Milestone 2 ($T_0+24$): Deliver first prototype.
\item Phase 3 ($T_0+24 \rightarrow T_0+36$) : Second protoype.
\item Milestone 3 ($T_0+36$): Deliver second prototype.
\item Phase 4 ($T_0+36  \rightarrow T_0+42$ : Final user evaluations.
\end{description}
